# Mörderspiel

Diese App soll dabei helfen, Mitspieler des Gruppenspiels [Mörderspiel](http://www.spielewiki.org/wiki/M%C3%B6rderspiel) mit den richtigen Mordzielen zu versorgen und den Spielverlauf zu verfolgen.
